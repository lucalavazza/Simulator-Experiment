yawning\_titan package
======================

The YAWNING TITAN package is broken into three broad components.

1.  :code:`yawning_titan.agents` - This contains classes and associated functions for creating non-learning agents
    within the environments proved by YAWNING TITAN
2.  :code:`yawning_titan.envs` - This contains the classes and associated functions for creating environments
3.  :code:`yawning_titan.experiment_helpers` - This contains a series of scripts to train and evaluate agents using
    the aforementioned components.
